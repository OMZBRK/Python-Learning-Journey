# ⚔️ RPG Battle Engine

## Description
A turn-based combat simulator featuring a Duel between two ninjas. This project focuses on state management and complex conditional logic.

## Key Concepts
- **Game Loop**: A `while` loop that maintains the battle until a win/loss condition is met.
- **Decision Logic**: An automated enemy that chooses its move based on available Chakra.
- **Error Handling**: Uses `try/except` to prevent the game from crashing if the player enters an invalid command or lacks energy.

## How to play
1. Run `python battle_engine.py`
2. Choose `1` for a Standard Attack.
3. Choose `2` for a Special Move (Costs 20 Chakra).
